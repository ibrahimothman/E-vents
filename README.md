# E-vents

A simple app to show some events using Strapi and Gridsome

## Running Locally

**Front-end:**

1. `npm install`
1. `npm run develop`
1. App will run on port `8080`

**Back-end:**

1. `npm install`
1. `npm run develop`
1. App will run on port `1337`
1. Go to [http://localhost:1337/admin](http://localhost:1337/admin) and create a new user, events and categories.
